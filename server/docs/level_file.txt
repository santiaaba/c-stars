# Determina la estructura del archivo que define el nivel.

Configuración general
---------------------
Campo				Bytes			Detalle
ataques			1				Cantidad de ataques
background		1				id background fondo
soundtrak		1				id música de fondo

Estructura del ataque
-------------------------------
Campo				Bytes			Detalle
cantShips		1				Cantidad de template definidas
time				2				momento en que arranca el ataque

Estructura del template de nave enemiga
---------------------------------------
Campo				Bytes			Detalle
type				1				tipo de nave
x					2				coordenada x de inicio
y					2				coordenada y de inicio
time				2				momento en que arranca la nave
movements		1				cantidad de movimientos

Estructura del movimiento
-------------------------
Campo				Bytes			Detalle
modulo			1				módulo del vector
direccion		1				dirección del vector
duracion			1				duracion del movimiento
